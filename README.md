# 🎯 Aim Trainer Web Game (Fast Trigger)

Bu proje, FPS oyuncularının reflekslerini, klik doğruluğunu (accuracy) ve "counter-strafing" takip yeteneklerini geliştirmek amacıyla saf JavaScript, HTML5 ve CSS3 teknolojileri kullanılarak geliştirilmiş web tabanlı bir antrenman simülatörüdür. 

Oyun, oyuncuların skorlarını küresel bir liderlik tablosunda (Leaderboard) gerçek zamanlı olarak yarıştırabilmesi için **Supabase (PostgreSQL)** backend altyapısı ile entegre çalışmaktadır.

---

## 🚀 Öne Çıkan Özellikler

- **Gelişmiş Hedef Mekanikleri:** Standart hareket eden ve küçülen hedeflerin yanı sıra, gerçekçi e-spor senaryolarını taklit eden **A-D Strafe (Counter-Strafe)** yapan akıllı hedefler.
- **Dinamik Çarpan Sistemi (Risk/Ödül):** Ardı ardına 10 başarılı vuruş (Combo) yapan oyuncular için **2X Puan Çarpanı** aktif olur. Ancak yapılan her iska (boşa tıklama) komboyu sıfırlar ve **-1 Ceza Puanı** uygular.
- **Gelişmiş Vuruş Algılama:** Hedeflerin tam merkezine yapılan vuruşlar **Headshot (+3 Puan)**, kenar vuruşları ise **Normal Hit (+2 Puan)** olarak hesaplanır.
- **Kullanıcı Kontrolleri:** Oyun esnasında dinamik olarak değiştirilebilen Crosshair (Nişangah) seçimi ve ses seviyesi (Volume Slider) kontrolü.
- **Zamanlayıcı ve Panik Butonları:** Maç ortasında gidişattan memnun olmayan oyuncular için skor kaydetmeden ana menüye dönmeyi sağlayan "Oyundan Çık" veya maçı sıfırlayan "Yeniden Başlat" mekanizmaları.

---

## 🛠️ Kullanılan Teknolojiler

- **Frontend:** HTML5, CSS3 (Custom Variables, CSS Animations), Pure JavaScript (ES6+, RequestAnimationFrame API)
- **Backend / Database:** Supabase (PostgreSQL), Row Level Security (RLS)
- **Audio API:** Web Audio element senkronizasyonu ile sıfır gecikmeli ses oynatma.

## 🔧 Kurulum ve Çalıştırma Rehberi

Projeyi kendi yerel ortamınızda çalıştırmak ve kendi veritabanınıza bağlamak için aşağıdaki adımları takip edin:

### 1. Projeyi Klonlayın
```bash
git clone [https://github.com/KULLANICI_ADINIZ/aim-trainer-web.git](https://github.com/KULLANICI_ADINIZ/aim-trainer-web.git)
cd aim-trainer-web
```

### 2. Ses Dosyalarını (Audio) Ekleyin
Depo boyutunu hafif tutmak ve telif haklarını korumak amacıyla proje içerisinde ses dosyaları (.mp3) bulunmamaktadır. Oyunun hatasız çalışması için kendi bulduğunuz veya kaydettiğiniz ses dosyalarını projenin ana dizinine (index.html ile aynı yere) aşağıdaki isimlerle eklemelisiniz:

sayim.mp3 : Oyun başlarken çalan 3-2-1 geri sayım (kalkış) sesi.

vurus.mp3 : Hedefe kenardan isabet edildiğinde çıkan normal vuruş sesi.

headshot.mp3 : Hedefin tam merkezine isabet edildiğinde çıkan özel ses.

iska.mp3 : Boşa tıklandığında (-1 puan) çıkan hata sesi.

kazandin.mp3 : Süre bittiğinde ilk 3'e girilirse çalan tebrik müziği.

kaybettin.mp3 : Süre bittiğinde ilk 3'e girilemezse çalan kaybediş müziği.

### 3. Supabase Veritabanı Kurulumu
Supabase üzerinde ücretsiz bir hesap oluşturun ve yeni bir proje başlatın.

Sol menüden SQL Editor bölümüne girin ve New Query butonuna basarak aşağıdaki SQL kodunu yapıştırıp çalıştırın:
```
CREATE TABLE public.leaderboard (
    id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    username text NOT NULL,
    score integer NOT NULL,
    created_at timestamp with time zone DEFAULT timezone('utc'::text, now()) NOT NULL
);
```
Güvenlik (RLS) Ayarı: Sol menüden Authentication -> Policies kısmına gelin. leaderboard tablosu için RLS'yi aktif hale getirin (Enable RLS). Ardından yeni bir policy oluşturup (Create policy from scratch), anonim kullanıcıların erişebilmesi için ALL operasyonuna izin verin ve hedef rolü anon olarak belirleyip USING ve WITH CHECK kutularına true yazarak kaydedin.

### 4. API Anahtarlarını Bağlama
Supabase panelinizde Project Settings -> API bölümüne gidin.

Project URL ve anon (public) key değerlerinizi kopyalayın.

Proje klasörünüzdeki script.js dosyasını açarak en üstteki ilgili alanlara kendi değerlerinizi yapıştırın:
```
const SUPABASE_URL = "KOPYALADIGINIZ_URL_ADRESI"; 
const SUPABASE_ANON_KEY = "KOPYALADIGINIZ_ANON_KEY";
```
### 5. Çalıştırın
Herhangi bir yerel sunucuya (localhost) ihtiyacınız yoktur. Ses dosyalarını ekleyip API anahtarlarınızı girdikten sonra index.html dosyasına çift tıklayarak oyunu tarayıcınızda oynayabilirsiniz!
